# Stroke Risk Prediction using Machine Learning

## Project Overview
This project predicts the likelihood of stroke based on patient health information using machine learning techniques.

## Objectives
- Analyze stroke-related health data.
- Build predictive machine learning models.
- Compare model performance.
- Deploy the model using Streamlit.

## Technologies Used
- Python
- Pandas
- NumPy
- Scikit-learn
- Matplotlib
- Streamlit

## Features
- User-friendly web interface.
- Real-time stroke risk prediction.
- Input patient health details.
- Display prediction results instantly.

## Project Workflow
1. Data preprocessing
2. Exploratory Data Analysis
3. Feature engineering
4. Model training
5. Model evaluation
6. Streamlit deployment
